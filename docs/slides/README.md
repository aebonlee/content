## 슬라이드 규칙

- `---` (가로줄)이 **슬라이드 경계**입니다. (Marp · Slidev · Gamma · Claude PPT 호환)
- 슬라이드 첫 줄 `##` = 슬라이드 제목, `-` = 본문 불릿.
- ` ```프롬프트 ``` ` 코드블록 = 그대로 복붙해 쓰는 **실습 프롬프트**(발표 시 화면 그대로 노출).
- `> 🗣️` = **강사 발표 노트**(슬라이드 본문엔 넣지 않아도 됨).

## Claude에게 줄 제작 프롬프트(예시)

> 아래 마크다운을 16:9 파워포인트 강의 슬라이드로 만들어줘.
> 슬라이드 경계는 `---`. 디자인은 아래 아트디렉션을 따르고, `> 🗣️`는 발표자 노트로,
> ` ```프롬프트 ``` ` 블록은 코드 스타일 박스로 강조해줘. 한 슬라이드에 너무 많은 글자를 넣지 말고
> 핵심 키워드 위주로, 설명은 발표자 노트로 빼줘.

### 아트디렉션 (사이트와 동일 팔레트로 통일)

- **무드**: 깊은 / 집중되는 / 신뢰가는. 청사진(blueprint)·모눈 노트·만년필 남색 잉크.
- **팔레트**
  | 역할 | HEX |
  |---|---|
  | 배경(Paper) | `#F4F6FB` |
  | 본문/제목(Ink·자정 남색) | `#14213D` |
  | 주조(Pigment·학술 블루) | `#1D4E89` |
  | 강조(Accent·형광 호박, CTA·형광펜만) | `#E8A33D` |
  | 섹션 배경(Wash) | `#DCE6F2` |
- **타이포**: 제목 **Noto Serif KR**(절제해서 크게) · 본문 **Pretendard**(행간 넉넉) · 수치/캡션 **JetBrains Mono**.
- **시그니처**: 핵심 한 문장에만 **형광펜(amber)** 밑줄. 남발 금지.
- **금지**: 알록달록 캐릭터 / 보라색 SaaS 그라데이션 / 과장된 뱃지·카운터.
